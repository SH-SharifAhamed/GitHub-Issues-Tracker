    - 1️⃣ What is the difference between var, let, and const?

    var পুরোনো ভাবে ভ্যারিয়েবল declare করতে ব্যবহার হয়।
    let এর মান পরে পরিবর্তন করা যায় |
    const এর মান পরিবর্তন করা যায় না।


    - 2️⃣ What is the spread operator (...)?

      Spread operator (...) ব্যবহার করে array বা object এর ভেতরের সব মান আলাদা করে ছড়িয়ে নেওয়া যায়।


    - 3️⃣ What is the difference between map(), filter(), and forEach()?

      map() নতুন array return করে,

      filter() শর্ত অনুযায়ী কিছু data নিয়ে নতুন array দেয়,
       
      forEach() শুধু loop চালায় কিন্তু কিছু return করে না।


    - 4️⃣ What is an arrow function?

      Arrow function হলো JavaScript এর ছোট ও সহজভাবে function লেখার একটি syntax।


    - 5️⃣ What are template literals?

      Template literals ব্যবহার করে backtick ` ` এর ভিতরে string লিখে সহজে variable বা expression ${} এর মাধ্যমে বসানো যায়।
